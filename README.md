<h3>Application Setup</h3>
<ul>
  <li>Clone or Download the repository</li>
  <li>cd plato/</li>
  <li>bundle install</li>
  <li>rake db:create</li>
  <li>rake db:migrate</li>
  <li>rake db:seed</li>
</ul>
